js数组方法

数组方法总结(一)

该篇文章部分内容参考大漠所写的系列文章，链接如下：https://www.w3cplus.com/javascript/array-part-5.html

js数组方法很多，一一描述太过于繁琐，所以只对我们日常编程中常用的方法进行整理总结。本篇所有代码均已在Chrome浏览器中测试过，可以使用快捷键Ctrl+Shift+J打开控制台，输入本篇代码实践一下。

concat()

该concat()方法用于合并两个或更多数组。此方法不会改变原有的数组，而是返回一个新数组。
返回的新数组中的元素包括原始数组的元素和concat()的每个参数。需要注意的一点是，如果这些参数中的任何一个自身也是一个数组，则连接的是数组的元素而不是数组本身。示例如下：

slice()

slice()方法与concat()方法类似，也是基于当前数组进行操作，返回新数组，并对原始数组没有任何影响。
slice()接受两个参数，分别为开始和结束位置。返回的新数组包含从开始到结束位置之间的所有数组元素，如果只指定了一个参数，则新数组包含从该参数开始一直到原始数组结束的所有元素。实例如下：

splice()

splice()方法是在数组中插入或删除元素的通用方法。与concat()和slice()不同，splice()会修改原数组，返回值是删除的数组项。主要用法如下:

  • 删除 : 可以删除任意数量的数组项
  • 插入 : 可以向指定位置插入任意数量的数组项
  • 替换 : 可以删除任意数量的数组项，且同时插入任意数量的数组项

删除

两个参数，第一个参数指定开始删除数组项的位置，第二个数是指删除数组的个数。若第二个参数不指定，则删除从开始位置到结束的所有数组项。示例:

由于删除的数组项为0，所以返回的数组为空数组

替换

参数与插入的参数相同，只是第二个参数变为你想删除的数组项数量

总结

总结了concat()、slice()和splice()三个方法。其中concat()和slice()不会影响原数组，会基于原数组创建一个新数组，而splice()方法会直接对原数组进行操作，并始终返回一个数组。
