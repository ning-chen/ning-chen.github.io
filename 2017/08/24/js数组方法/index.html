js数组方法总结(二)

JavaScript数组提供这些方法来模拟栈和队列。如果对栈和队列的基础知识不太了解的话，可以看看大漠的这篇文章https://www.w3cplus.com/javascript/array-part-3.html，很简单易懂

栈方法

数组可以表现得就像栈一样，后者是一种可以限制插入和删除项的数据结构。栈是一种LIFO(Last-In-First-Out,后进先出)的数据结构，也就是最新添加的项最早被移除。而栈中项的插入(叫做推入)和移除(叫做弹出)，只发生在一个位置——栈的顶部。ECMAScript 为数组提供了 push() 和 pop() 方法，以便实现栈的行为。摘自《JavaScript高级程序设计》

队列方法

队列数据结构的访问规则是LIFO(后进先出),而队列数据结构的访问规则是FIFO(FIrst-In-First-Out,先进先出)。队列在列表的末端添加项，从列表的前端移除项。 摘自《JavaScript高级程序设计》

unshift()

unshift()方法是向数组前端添加一个或多个元素，并返回新的长度。

由于unshift()在开头添加元素，每添加一个元素，都要把现有的元素往下移一个位置，所以效率低。当对大数组进行处理时，可以使用数组的reverse()方法，该方法将一个数组反转。然后将要放进数组的元素用push()方法添加，再执行一次reverse()，就与unshift()的效果相同。有人说这种办法比直接使用unshift()繁琐，所以效率也不会太好。我们用数据说话，事实胜于雄辩！具体测试参考本篇文章，链接奉上

总结

  • push()方法可以在数组的末属添加一个或多个元素，返回修改后的数组长度
  • shift()方法把数组中的第一个元素删除，返回删除的元素
  • unshift()方法可以在数组的前端添加一个或多个元素，返回修改后的数组长度
  • pop()方法把数组中的最后一个元素删除，返回删除的元素
  • 四个方法都直接修改原数组
